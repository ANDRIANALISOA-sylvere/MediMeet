Description des Champs
Users : Gère les informations de base des utilisateurs (patients et médecins).
Patients : Stocke des informations spécifiques aux patients, y compris les antécédents médicaux.
Doctors : Contient des informations spécifiques aux médecins, telles que la spécialité et les horaires de disponibilité.
Appointments : Gère les informations de prise de rendez-vous.
Notifications : Stocke les notifications envoyées aux utilisateurs.
Messages : Gère les messages échangés entre les patients et les médecins.
Teleconsultations : Contient les informations liées aux consultations vidéo.
Payments : Gère les informations de paiement pour les consultations.
Reviews : Stocke les avis et évaluations des médecins par les patients.
MedicalRecords : Gère les dossiers médicaux des patients.
Resources : Gère les ressources disponibles pour les consultations.
Reports : Contient les rapports et analyses statistiques.
